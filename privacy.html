Privacy Policy

1. Collected Data: We store only minimal data: TikTok Open ID and basic video statistics. Profile info like nickname and avatar is used temporarily but not stored.

2. Purpose: Data is used only to display analytics to the authenticated user. No personal or sensitive data is processed.

3. No Sharing: We do not share any data with third parties. All data is used internally for analytics display only.

4. No Extra Permissions: We do not collect geolocation, contact info, or payment details. Only essential TikTok Login Kit scopes are used.

5. Data Security: Data is stored securely and used only for the described purpose. Users may request data deletion at any time via the bot.
